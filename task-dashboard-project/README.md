# TaskFlow Dashboard

A clean and beginner-friendly task management dashboard built using **HTML, CSS, and JavaScript**.

## Features

- Add tasks with title, priority, and due date
- Mark tasks as completed
- Delete tasks
- Search tasks
- Filter by all, pending, and completed
- Clear completed tasks
- Saves data in browser using `localStorage`
- Modern glassmorphism UI

## How to Run

### Option 1: Open directly
Just open `index.html` in your browser.

### Option 2: Use VS Code Live Server
1. Open the project folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Click **Open with Live Server**
